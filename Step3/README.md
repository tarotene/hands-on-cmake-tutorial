# Step 3: Adding Usage Requirements for a Library

## Exercise 1 - Adding Usage Requirements for a Library

### Build and Run / Exercise 1

- Prepare

    ```sh
    mkdir -p Step3_build
    cd Step3_build
    ```

- Configure

    ```sh
    cmake ../Step3
    ```

- Build

    ```sh
    cmake --build .
    ```

- Run

    ```sh
    ./Tutorial
    ```

    > Expected output:
    >
    >   ```plaintext
    >   ./Tutorial Version 1.0
    > Usage: ./Tutorial number
    >   ```

- Disorganize

    ```sh
    cd ../
    ```
