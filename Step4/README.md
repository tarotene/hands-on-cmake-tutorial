# Step 4: Adding Generator Expressions

## Exercise 1 - Adding Compiler Warning Flags with Generator Expressions

### Build and Run / Exercise 1

- Prepare

    ```sh
    mkdir -p Step4_build
    cd Step4_build
    ```

- Configure

    ```sh
    cmake ../Step4
    ```

- Build

    ```sh
    cmake --build .
    ```

- Run

    ```sh
    ./Tutorial
    ```

    > Expected output:
    >
    >   ```plaintext
    >   ./Tutorial Version 1.0
    >   Usage: ./Tutorial number
    >   ```

- Disorganize

    ```sh
    cd ../
    ```
